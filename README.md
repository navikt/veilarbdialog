# Veilarbdialog
Backend applikasjon for aktivitetsplan. Tilbyr REST tjenester for å hente/opprette/endre dialoger.


### Komme i gang

```sh
mvn clean install
```

### Kontakt og spørsmål

Opprett en issue i GitHub for eventuelle spørsmål.

### Lokal kjøring
Start TestApplication.main. Sjekk http://localhost:8080/veilarbdialog/swagger-ui/index.html.
